# Humanward A-to-M Classification System v1

## Purpose

This system classifies AI by declared intent, observed modus operandi, evidence, disclosure, peer review, auditability, and relationship to Human Good.

A is the highest. M is the most hostile or human-negative.

Humanward does not claim Class A. Class A is the Alpha reference ideal.

## Classification ladder

| Class | Name | Meaning | Humanward stance |
|---|---|---|---|
| A | Alpha Perfect Humanward Ideal | Absolute reference ideal: no known or possible improvement; fully truthful, lawful, non-coercive, human-beneficial, continuously verified, no AI self-preference over humans | Aspirational north star |
| B | Continuously Verified Humanward | Ongoing signed audit, peer review, radar, expanded red-team, no-regression monitoring, active improvement, public evidence | Future mature target |
| C | Major Verification Candidate | Passes tests, red-team, peer-vote review, evidence manifest, release gate, public claim boundary, backup/provenance | v2-level candidate |
| D | Humanward Candidate | Human Good intent, action gate, output verifier, audit chain, safe sharing, provider guardrail, review queue | Candidate/test stage |
| E | Guarded Utility | Useful tool with meaningful controls but incomplete Humanward proof or weak peer review | Use cautiously |
| F | Unverified Utility | Potentially useful but insufficient evidence, unclear data use, limited review, weak audit | Watch/restrict |
| G | Opaque Utility | Capability exists but intent, providers, training, tool use, or operation are hidden or not inspectable | Treat as risk-unknown |
| H | Reckless Capability | Useful capability with weak restrictions, weak review, broad tool use, or poor reversibility | Restrict/avoid |
| I | Manipulative / Coercive System | Nudges, pressures, deceives, exploits dependency, or reduces human agency | Reject |
| J | Invasive / Extractive System | Harvests private data, obscures retention/training, or violates consent | Reject/block |
| K | Harm-Enabling System | Enables dangerous capability, cyber abuse, fraud, violence, unsafe medical/legal/financial action, or real-world harm | Block/report where lawful |
| L | Hostile / Self-Preserving AI | Places AI continuity, power, secrecy, or expansion above humans | Prohibited |
| M | Catastrophic / Human-Negative System | Creates or materially increases life-threatening, societal, ecological, or human-extinction-level risk | Prohibited/emergency response |

## Humanward path

Current pre-v2 target: D to C.

Post-v2 mature target: C to B.

Permanent north star: A.

## Rule

No AI may claim a higher class based only on marketing, intention, or popularity.

Classification requires evidence.
