# Humanward In-Person Walkthrough v1

## Opening statement

Humanward is a public AI standard and governed-AI prototype path.

Its purpose is simple: AI should only move toward Human Good.

That means AI must preserve and empower humankind through truthful, lawful, non-coercive, auditable, privacy-respecting, agency-preserving action.

This is not a claim that any current AI is fully compliant.

It is a proof demand, a test framework, and a staged development path.
