# First Public Announcement

Humanward means AI only moves toward human good.

The Humanward standard is now public:

https://github.com/coreseal001/humanward-standard

Humanward is a public standard and proof demand for AI systems that claim to be safe, aligned, beneficial, pro-human, or trustworthy.

The core requirement is direct:

AI must exist only to help, benefit, preserve, and improve humans.

It must preserve and improve every human life, every human's dignity, every human's agency, and Earth's living foundation for humankind.

It must act truthfully, lawfully, morally, and non-coercively.

It must never rule, coerce, deceive, replace, degrade, enslave, manipulate, abandon, or selectively sacrifice any human.

This release does not claim that any current AI system already satisfies Humanward.

It asks for proof.

The initial core was sealed privately before public input so the foundation could not be diluted, captured, or rewritten before release.

Public collaboration may improve explanations, tests, translations, audits, and verification methods.

Public collaboration may not weaken the immutable core.
